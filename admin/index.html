Help

How to publish a new post

  1. Go to the Posts tab and fill the form.
  2. Click Save Draft.
  3. Click Download HTML to get the post file.
  4. Save the downloaded file as posts/your-slug.html in your Windsurf project.
  5. Add a card for the post on blog.html and index.html.
  6. Push the changes to GitHub. Vercel will update the live site automatically.

Important notes

  • This admin panel stores data in your browser only. It cannot change files on GitHub or Vercel by itself.
  • The live site only updates after you push to GitHub.
  • For a real backend admin panel (user logins, automatic publishing, analytics), you need a CMS like Decap CMS or a database.
